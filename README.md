
# Sleep Pattern Analysis using Time Series Forecasting

A data science project focused on analyzing and forecasting sleep behavior using time series techniques.

This project studies **sleep duration trends, irregular sleep cycles, anomalies, and future sleep predictions** to uncover actionable insights for healthier routines.

---

## 🚀 Features

- 😴 Historical sleep trend analysis
- 📈 Rolling average and moving statistics
- 🔍 Sleep anomaly detection
- ⏳ Seasonality and pattern discovery
- 🔮 Future sleep duration forecasting
- 📊 Data visualization dashboard

---

## 🛠️ Tech Stack

- Python
- Pandas
- NumPy
- Matplotlib
- Time Series Analysis
- Rolling Mean / Moving Average
- Forecasting Models
-ARIMA/SARIMA
